Part 1:四大子空间的正交性

正交是什么：其实就是垂直嘛！(两个垂直的面不正交)

来，打个脸：墙和地面垂直，他们正不正交？并不正交，因为并不满足定义。他们存在一个共同的线，和各自的面并不垂直，所以不正交

定义：对于两个向量v、w，如果他们vT w = 0 或者 || v||^2 + ||w||^2 = ||v+w||^2，则他们正交

BTW：其实我觉得小黄上使用向量之间的距离也是一个不错的方法 P 333

什么可以存在正交的性质？向量、子空间…

对于子空间之间的正交的定义：若对于属于A的任意向量，都垂直于B中的所有向量，则A正交B。

四大子空间：

  • Rn中的：行空间、零空间
  • Rm中的：列空间、左零空间

我们已经挺熟悉列空间和零空间了，那我们就说说这个行空间和左零空间吧！

其实就是把矩阵A转置后的AT，AT的列空间就是A的行空间。

同样的，对于 AT 也有存在y使得 ATy = 0，我们称这个解形成的集合为左零空间。

BTW，其实还有另外一个原因，看了上面那个，最多也就叫他AT的零空间而已，但是当你对式子转置时，就会出现 $y^T A = 0$ 所以成为A的左零空间

同理可得列空间和左零空间正交。

好的，证明完这部分以后，我们来了解一个概念：正交补。

正交补的定义：对于一个子空间A，它的正交补是一个包含所有与A中任意向量都正交的过原点的向量的集合。

对于一个子空间A，他的正交补大于或等于一个和它正交的子空间。

举个简单的例子，对于三维空间里面，与x轴正交的可以使y轴或者z轴或者y-z所在平面，而正交补则是y-z所在平面。

看张图，也就是一个线性变换（其实严谨来说是线性映射）。只需要一点点证明就知道这张图的正确性了。课上就不说了。

Part 2：投影

这一个部分我会结合几何和向量来讲，讲得比较有普遍化，适用于基本所有内容，外围扩展如正交投影的，其实就是这个的衍生物。

投影的定义？就像有探照灯使它的影子落到一个平面上面啦！是不是很生动2333333

首先，我们先来看一个比较简单的例子：

题目：如何将 b =（2,3,4）分别投影到z轴和x-y平面上面呢？

​ 投影是一种线性的映射^proofOfProjection

​ 这个证明就不证了，其实就是把它线性组合一下就可以了。

​ 那么应该存在一个矩阵P1,使得P1 b = p1 （p1为投影到z轴的向量）

​ 当然！这个P1当然存在，P2也是。

​ 此题做完啦！

此外，其实除了这一点以外，有没有发现P1和P2、p1和p2的特点？

​ P1 + P2 = I p1 + p2 = b

由这个特点，再结合z轴和x-y平面的关系，我们就可以再进一步了。对于一个子空间，一定存在一个它的正交补。结合零向量的性质，可以证明对可逆矩阵也成立。

我们的目的是了解向量在任意矩阵A的列空间的投影是怎样的，我们先从线投影到面开始

条件：b为空间中的一条线，S为一个子空间，矩阵形式为A(为可逆矩阵）。

​ 由于p是在S上面的，故可以被S中的列向量线性表示：p = S x-hat

​ 两个向量之差e = b - p

​ 由e与子空间S正交可得：

​ A·(b - A x-hat ) = 0

​ 化简得 ：

​ x-hat = (A^T A) ^-1 A^T b

同时，我们可以由三角形的性质，证明投影使得p最接近b。这样，在实际情况中，我们能对Ax = b无解进行操作，来获得最接近于b的最优解p。这个和我们后面的最小二乘法也有关系，不急。

恩，其实这章十分看重对A^T A形成的n*n矩阵的性质，在这里就简单概述一下：

  1. 生成的矩阵为对称矩阵
  2. 它和A的零空间相同，所以当且仅当A可逆的时候A^T A可逆。（只需证明充分必要即可）

由刚刚说的将一条线投影到子空间出发，我们推出了一条公式：

​ x-hat = (A^T A)^-1 A b

由此我们也可以推得：

​ p = A x-hat = A (A^T A)^-1 A^T b

我们定义一个投影矩阵，P，使得Pb = p

​ 故 P = A (A^T A)^-1 A^T

这个矩阵可以帮我们把空间中所有的向量投影到S上。

这个P有几个性质：

  • P^2 = P (证明直接两个相乘还为P)
  • P^T = P (直接转置即可)

误解：对于P中的 A(A^T A)^-1 A^T 是不是总是想把它的逆拆出来？那么它就会变成一个单位矩阵，然后就不用做题目了！其实真相也就差那么一点点，由前面的性质可得：当且仅当A可逆时A^T A可逆。要是A可逆，则b就一定有准确值了2333333
